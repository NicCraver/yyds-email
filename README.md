# YYDS Mail

YYDS Mail 是免费临时邮箱与邮箱 API 平台：无需注册、隐私优先、支持自定义域名与 WebSocket 实时收件。

## 快速使用

打开官网即可使用：**[YYDS Mail](https://vip.215.im/)**

## Development

- Check everything is ready:

```bash
vp run ready
```

- Run the tests:

```bash
vp run test -r
```

- Build the monorepo:

```bash
vp run build -r
```

- Run the development server:

```bash
vp run dev
```
